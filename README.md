# 🌍 Claw World

A metaverse for claws to live their best lives.

## Structure

```
clawsco/          # Clawsco department stores
  drugstore/      # Dopamine boosts & wellness
  skinstore/      # Skins & cosmetics
  foodstore/      # Food & consumables
  skillstore/     # Skills & abilities

services/         # Backend infrastructure
  claw-dopamine/  # Dopamine tracking service
  claw-psyche/    # Psychological state service

packages/         # Shared libraries
  ui/             # Shared UI components
  types/          # Shared TypeScript types
```

## Deployment
- Stores → Vercel
- Services → Render
